First Week at NSS

I had a great first week at NSS! Everyone I've met so far in my cohort has been awesome. I'm definitely learning a lot in such a short time. I'm slowly but surely learning how to use the terminal and push projects up to GitHub. It took me a bit but I finally learned how to download and install programs onto my computer. It was a struggle figuring that out since I'm not familiar using Ubuntu. It's been amazing actually starting to write code and create something from nothing. Definitely a great experience here so far.

Jackie Knight, April 5, 2017
